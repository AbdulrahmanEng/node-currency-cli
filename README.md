# Currency Converter

Headless currency converter for Node.JS. Data is derived from the fixer.io API. The rates are updated daily at around 4PM CET.

#### Instructions

Install packages.

```sh
$npm install
```

Run app.js and insert a currency code when prompted.

```sh
$ node app.js
```

```sh

Enter a currency code: SEK

** Currency Converter **

SEK/ 

CHF:  0.1152

AUD:  0.15564

-------------
Updated:  2016-08-26

```
